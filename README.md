# Flame🔥

![C++](https://img.shields.io/badge/language-C++-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Header Only](https://img.shields.io/badge/type-header--only-orange)

A lightweight, SIMD-accelerated math library designed for game engines and real-time applications.

---

## ✨ Features

- Header-only design
- SIMD optimized (SSE/AVX)
- Vector, Matrix, Quaternion support
- Ray and AABB intersection
- Camera utilities

---

## 📦 Installation

Just include the header:

```cpp
#include <flame/flame.h>
